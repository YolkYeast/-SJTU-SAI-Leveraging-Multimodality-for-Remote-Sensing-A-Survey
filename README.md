## üéØ Project Overview

This repository serves as a comprehensive collection of references for the final project on **"Leveraging Multimodality for Remote Sensing"**. The project explores the evolution and application of multimodal approaches in remote sensing, from traditional multimodal vision techniques to modern Multimodal Large Language Models (MLLMs).

### Key Topics:
- **Multimodal Vision**: Traditional and modern approaches to combining multiple data modalities in remote sensing
- **Multimodal Large Language Models**: Recent advances in MLLMs for understanding and processing remote sensing data
- **Remote Sensing Applications**: Practical applications across various domains (agriculture, urban planning, disaster management, etc.)

## üìù How to Add References

To maintain consistency when adding new references, please follow this format:

### For Papers/Articles:
```markdown
| [Paper Title] | Year | [Source Website](URL) | Type | Brief description or key contribution |
```

### For Datasets:
```markdown
| Dataset Name | Year | [Source Website](URL) | Description | Additional notes (size, format, etc.) |
```

### Example Entry:
```markdown
| Vision Transformers for Remote Sensing | 2023 | [arXiv](https://arxiv.org/abs/2103.14030) | Paper | Applies ViT to satellite image classification |
```

### Tips:
- Use descriptive titles
- Include the publication year
- Provide direct links to the source (paper, dataset, code repository)
- Add brief notes about why this reference is relevant
- Categorize the reference under the appropriate section
